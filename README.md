# bfh-advanced-cpp-cheatsheet
Cheatsheet for the elective course "Advanced C++" at the Bern University of Applied Sciences.

## Required Software
- pdflatex
- BibTeX
- GNU Make

Otherwise just modify the Makefile to your preferences.

## Building
### Compilation
```shell
$ make
```

### Removal of Auxiliary Files
```shell
$ make clean
```
